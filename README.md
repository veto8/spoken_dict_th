## Project Overview

Welcome to the Open Spoken Dictionary (OSD) project! This repository contains a collection of over 40,000 MP3 audio files, each representing the spoken pronunciation of a thai word. Our goal is to create a comprehensive, freely available resource for language learners, developers, and researchers.

We believe that access to high-quality audio pronunciations is crucial for language acquisition, accessibility tools, and various other applications. By making this dataset open source under the GNU GPL v3 license, we aim to ensure that the data and any derivative works remain free and accessible to all.

**Why contribute?**

*   **Help language learners:** Provide a valuable resource for improving pronunciation and comprehension.
*   **Support accessibility:** Enable the development of tools for individuals with visual impairments or learning disabilities.
*   **Advance research:** Facilitate research in speech recognition, natural language processing, and phonetics.
*   **Be part of a community:** Join a collaborative effort to build a comprehensive spoken dictionary.
*   **Uphold Software Freedom:** Contribute to a project that prioritizes user freedom and community-driven development.
